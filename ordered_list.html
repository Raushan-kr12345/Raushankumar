HTML Ordered Lists

An ordered list created using the "ol tag" element ,and each list item starts with the "li tag" element. Orderd lists are used when the order of the list's items is importantant.

The list items in unordered lists are marked with numbers.

Attributes type of ordered list

  • A,B,C,D.....;

  • 1,2,3,4.....;

  • i,ii,iii,iv......

  • a,b,c,d.......
