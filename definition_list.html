HTML Definition Lists

An definition list is a of items with a description or definition of each item.

The description list is created using "d1 tag" element . The "d1 tag" element is used in conjuction with the " dt tag" element which specify a term and the "dd tag" element which specify the term's defination

Browsers usually render the defination lists by placing the terms and definition in separate lines, where the terms definitions are slinghtly indented.
